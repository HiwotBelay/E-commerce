# 🛒 My amazing E-commerce Project

A powerful, scalable e-commerce web app inspired by Amazon – built using **Next.js**, **React**, **Node.js**, **Express**, **Firebase Realtime Database**, **PostgreSQL**, and **Prisma**.

## 🚀 Features

- ✅ Smooth, responsive UI with clean navigation
- ✅ Real-time cart and order updates (via Firebase)
- ✅ Product filtering (by category & price range)
- ✅ User authentication (signup, login, logout)
- ✅ Newsletter subscription section
- ✅ Product categorization & review system
- ✅ Admin vs. user role-based access
- ✅ Dynamic cart & user notifications

---

## 🧠 Tech Stack

- **Frontend:** React, Next.js
- **Backend:** Node.js, Express
- **Database:** Firebase Realtime DB, PostgreSQL
- **ORM:** Prisma
- **Authentication & State:** Firebase Auth, Context API
